# 🛡 CyberScan - Vulnerability Scanner

CyberScan is an advanced vulnerability scanner built using Python and Flask.  
It detects open ports, analyzes web security headers, and provides a clean cyber-themed UI.

---

## 🚀 Features
- 🔍 Multi-threaded port scanning
- 🌐 Web vulnerability detection
- 🔐 Security headers analysis
- 🎨 Responsive UI (Cyber theme)
- ⚡ Fast scanning using concurrency

---

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- ThreadPoolExecutor

---

## ▶️ How to Run

```bash
pip install flask requests
python app.py
